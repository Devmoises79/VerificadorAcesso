🔐 Verificador de Acesso (C# Console)

Projeto simples em C# que simula um sistema de controle de acesso via terminal. O programa solicita nome e idade do usuário e aplica regras de validação baseadas em lista de nomes autorizados e idade mínima.

⚙️ Funcionalidades
- Leitura de dados via console
- Validação de nome em lista fixa (Joao, Maria, Pedro)
- Verificação de idade mínima (≥ 18 anos)
- Controle de acesso com lógica condicional

🧠 Conceitos aplicados
- Estruturas condicionais (if / else)
- Operadores lógicos (&&, ||)
- Entrada e saída de dados no console
- Validação de regras de negócio
- Programação estruturada em C#

🚀 Como executar
Abra o terminal e execute:

dotnet run

🛠️ Tecnologias
- C#
- .NET Console Application

👨‍💻 Objetivo
Projeto criado para prática de lógica de programação e controle de fluxo, simulando um sistema básico de autenticação.
